# TaskFlow — Консольдік Тапсырмалар Менеджері

## 🎯 Жобаға шолу

**TaskFlow** — бұл күнделікті тапсырмаларды тиімді жоспарлау және басқару үшін жасалған C++ тіліндегі консольдік қосымша. Жоба құрылымдық деректер, динамикалық массивтер және пайдаланушы енгізуімен жұмыс істеу сияқты бағдарламалаудың базалық және кеңейтілген концепцияларын көрсетеді.

---

# 🚀 Мәселе және шешім

## Мәселе

Оқу процесі кезінде студенттерге дедлайндар мен оқу тапсырмаларының басымдықтарын бақылау қиын. Қазіргі шешімдердің көбі артық функцияларға толы немесе интернетке тәуелді.

## Шешім

**TaskFlow** — күрделі интерфейстерге алаңдамай, тапсырмаларды ыңғайлы түрде ұйымдастыруға мүмкіндік беретін жеңіл әрі жылдам құрал. Толығымен оффлайн жұмыс істейді және бірден іске қосылады.

---

# ✨ Негізгі мүмкіндіктер (Features)

* ➕ **Тапсырма қосу:** Атауы мен басымдығын (Жоғары / Орташа / Төмен) көрсету арқылы жаңа тапсырма құру
* 📋 **Тізімді көру:** Барлық ағымдағы тапсырмаларды кесте түрінде түстік индикациямен көрсету
* ✅ **Статусты басқару:** Тапсырманы орындалды деп белгілеу ([ ] → [✓])
* 🗑️ **Тапсырманы жою:** Қажетсіз немесе ескірген тапсырмаларды өшіру
* 📊 **Статистика:** Орындалу прогресі мен басымдықтар бойынша бөліністі көрсету
* 🎨 **Интерфейс:** Түс арқылы ерекшеленген, қолдануға ыңғайлы мәтіндік мәзір

# 🛠️ Техникалық стек

* **Тіл:** C++ (C++17/20 стандарты)
* **Компилятор:** GCC 16.1.0 (MinGW-W64) немесе MSVC

## Кітапханалар:

* `<vector>` — тапсырмалар тізімін динамикалық сақтау үшін
* `<string>` — мәтіндік деректерді өңдеу үшін
* `<iostream>` — пайдаланушымен өзара әрекеттесу үшін
* `<windows.h>` — UTF-8 кодировкасын және консольді баптау үшін
* `<iomanip>` — форматталған шығару үшін

---

# 🏗️ Код архитектурасы

Жоба `struct Task` құрылымы негізінде құрылған. Бұл құрылым тапсырманың барлық атрибуттарын бір объектіге біріктіреді:

```cpp
struct Task {
    string name;           // Тапсырма атауы
    string priority;       // Басымдық (Жоғары, Орташа, Төмен)
    bool isCompleted;      // Орындалу статусы
    int id;                // Бірегей идентификатор
};
```

## Негізгі компоненттер:

* Деректер құрылымы: `vector<Task>`
* Өңдеу циклі: `while(true)` мәзірімен
* Командаларды өңдеу: `switch-case`

## Модульдік функциялар:

* `addTask()` — тапсырма қосу
* `displayTasks()` — тапсырмалар тізімін көрсету
* `completeTask()` — орындалды деп белгілеу
* `deleteTask()` — тапсырманы жою
* `showStatistics()` — статистиканы көрсету

## GCC (MinGW) арқылы:

```bash

```

## MSVC арқылы:

```bash
cl /EHsc /std:c++17 taskflow.cpp
```
